Toolkit Asisten Lab Elektro ini adalah program berbasis bahasa C yang dirancang untuk membantu perhitungan dasar pada praktikum elektronika.
Program ini menyediakan kalkulator rangkaian listrik serta fitur konversi bilangan dalam satu menu terpadu berbasis teks.

Tujuan dari program ini dibuat seperti pada kalkulator umumnya yaitu sebagai alat dalam membantu melakukan perhitungan dengan beberapa fitur fitur yang ada mulai dar kalkulator ohm, daya listrik, resistor seri, dan resistor parallel. Dalam program juga ada fitur yang digunakan untuk melakukan konversi bilangan dari bilangan decimal ke basis bilangan lain,

Fitur Program
Verifikasi Rangkaian Elektrik
1. Kalkulator Hukum Ohm  
   Menghitung Tegangan (V), Arus (I), atau Hambatan (R)  
   Rumus:  
   - V = I × R  
   - I = V / R  
   - R = V / I  

2. Kalkulator Daya Listrik 
   Menghitung daya listrik berdasarkan tegangan dan arus  
   Rumus:  
   - P = V × I  

3. Kalkulator Resistor Seri
   Menghitung hambatan total rangkaian resistor seri  
   Rumus:  
   - Rtotal = R1 + R2 + ... + Rn  

4. Kalkulator Resistor Paralel
   Menghitung hambatan total rangkaian resistor paralel  
   Rumus:  
   - 1/Rtotal = 1/R1 + 1/R2 + ... + 1/Rn  

---

Konversi
5. Konversi Desimal ke Basis Lain
   Mengonversi bilangan desimal ke beberapa sistem bilangan lain, yaitu:
   Biner (basis 2)
   Oktal (basis 8)
   Heksadesimal (basis 16)
   Fitur ini memungkinkan pengguna memilih basis tujuan (2–16) sesuai kebutuhan.

6. Konversi Biner ke Desimal
   Mengubah bilangan dalam sistem biner (basis 2) menjadi bilangan desimal.
   Program akan melakukan validasi input agar hanya menerima digit 0 dan 1.

7. Konversi Oktal ke Desimal
   Mengonversi bilangan oktal (basis 8) ke bilangan desimal.
   Input divalidasi agar hanya menerima digit 0 sampai 7.

8. Konversi Heksadesimal ke Desimal
   Mengubah bilangan heksadesimal (basis 16) ke bilangan desimal.
   Program mendukung digit angka (0–9) dan huruf (A–F atau a–f).
